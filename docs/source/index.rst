Welcome to pyROTMOD's documentation!
=================================
**pyROTMOD** is a mass modelling program that can take galfit models, tirific models, and mass profiles in a column based text file.
It uses galpy to model the mass distributions and  lmfit or numpyro to fit the rotation curves with mcmc methods.
